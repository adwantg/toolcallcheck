# Security Policy for agent-test

## Supported Versions

Only the current major version is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.x.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a vulnerability, please do NOT report it on the public issue tracker.

Instead, please email the maintainer directly at: `workwithgoutam@gmail.com`

We pledge to:
1. Acknowledge your report within 48 hours.
2. Provide a timeline for a patch.
3. Release a security advisory once the patch is published.

## Bug Bounty

Currently, we do not operate a bug bounty program, but we will publicly credit any reporter who responsibly discloses a verified vulnerability.
